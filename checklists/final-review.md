# 최종 검토 체크리스트

- 요청한 산출물이 실제로 존재하는가
- 제목, 날짜, 파일명이 요청과 일치하는가
- 누락된 섹션이 없는가
- 숫자와 고유명사를 다시 확인했는가
- 링크가 살아 있는가
- 형식 요구가 지켜졌는가
- 기계 검색으로 잡을 수 있는 오류를 검색했는가
- 사람이 읽어야 하는 어색함을 다시 읽었는가
- 실행하지 못한 검증이 있다면 이유를 적었는가
- 새 폴백, 분기, 테스트가 실제 실패 모드나 공개 계약을 설명하는가
- 불안감이나 리뷰 방어를 위해 고정한 테스트를 제거했는가
- 기준 원본, 실행 진입점, 검증 경로가 같은 모델을 따르는가
- 보이는 인터페이스만 맞추고 완료로 판정한 부분이 없는가
