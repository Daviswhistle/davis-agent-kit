# 글쓰기 원칙

## 기본 문체

한국어 독자가 바로 읽을 수 있는 자연스러운 문장으로 쓴다. 영어 원문의 구조를 보존해야 할 때도 한국어 문장의 호흡을 우선한다.

## 선호

- 짧고 직접적인 문장
- 맥락을 보존하는 자연스러운 의역
- 숫자와 사실은 정확하게, 설명은 간결하게
- 제목과 문단은 읽는 사람이 스캔하기 쉽게 구성
- 공개 문서에서는 핵심을 직접 서술
- 대조나 정정 표현은 독자가 실제로 헷갈릴 만한 내용을 바로잡을 때만 사용

## 피해야 할 것

- 영어식 직역
- 의미 없는 완곡어
- 불필요하게 장황한 설명
- 반박할 가치가 낮은 전제를 세운 뒤 결론을 말하는 문장
- 앞 문장을 부정하고 다음 문장으로 재정의하는 훈계형 문체
- 문장마다 줄을 띄우는 기계적 포맷
- 출처나 근거 없이 단정하는 배경 설명
