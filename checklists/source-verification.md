# 원자료 검증 체크리스트

- 원자료 위치를 확인했는가
- 원자료 날짜를 확인했는가
- 최신 정보가 필요한 항목인지 판단했는가
- 최신성이 필요한 경우 웹 검색 또는 공식 출처를 확인했는가
- 숫자, 사람 이름, 회사명, 제품명, 이벤트명을 대조했는가
- 원문에 없는 배경 설명을 추가했다면 근거를 기록했는가
- 숫자나 지표를 해석할 때, 분모(전체 집합)가 현실에서 대상이 존재하고 유효했던 기간인지 먼저 검증했는가
- 숫자가 무엇을 집계한 것인지 대상 집합, 기간, 단위, 누적·증분 기준을 확인했는가
- 관측값, 추정치, 전망, 시나리오, 모델 결과를 구분했는가
- 제안, 의견 수렴, 표결, 최종 결정, 시행, 실제 효과를 구분했는가
- 공식 통계·규제 문서, 논문, 기업 발표, 스트레이트 보도, 전문가 논평처럼 출처의 성격을 구분했는가
- 제목, 요약, 수치 카드가 원자료보다 강한 확정성이나 인과를 만들지 않는가
