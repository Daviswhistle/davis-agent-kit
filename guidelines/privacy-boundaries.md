# 공개/비공개 경계

## 공개 가능

- 일반화된 작업 절차
- 품질 체크리스트
- 공개 문서 번역 방식
- 공개 가능한 템플릿
- 민감 정보가 제거된 예시
- 투자 판단 방식
- 개인 선호와 작업 철학
- 반복 피드백에서 도출한 품질 기준
- 공개해도 되는 작업 사례와 의사결정 기록

## 비공개 유지

- 개인 계정, 토큰, 키
- 비밀번호와 인증 정보
- 비공개 원문 전문
- 원문 저작권 문제가 있는 긴 전문
- 제3자 민감정보
- 계약상 비공개 자료
- 사용자가 별도로 비공개로 지정한 메모

## 원칙

투자 판단이나 개인 선호도 기본적으로 공개 가능하다고 본다. 다만 공개 전에 비밀값, 저작권, 제3자 정보, 계약상 비공개 여부를 확인한다.
